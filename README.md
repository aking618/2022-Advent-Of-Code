## How to run

Go to the folder of the day you want to run and run the following command based on the language used.

### Javascript

```bash
index.html
```

### Java

```bash
javac *.java
java Driver
```

### PHP

```bash
php solutionOne.php
php solutionTwo.php
```

### Python

```bash
python3 main.py
```

### Dart

```bash
dart main.dart
```
